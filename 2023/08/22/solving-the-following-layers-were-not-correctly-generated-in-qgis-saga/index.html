Solving 'the following layers were not correctly generated' in QGIS SAGA

August 22, 2023

I was working on a map recently and tried to use SAGA in QGIS to make a multilevel b-spline raster from some scattered vector points. (More to come tomorrow!) However, I kept getting the vague error “The following layers were not correctly generated.”

As I experimented with the data, I discovered two circumstances that produce this error:

  • Attempting to interpolate a attribute with a space (or possibly other non-alphanumeric characters) in the name
  • Attempting to interpolate a non-numeric attribute

So if you’re encountering this, check that you’re interpolating a numeric attribute with a short alphanumeric name.
